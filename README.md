

# ECOM, E-Commerce Software

Source code from open-source project laracom
- See full [documentation](https://shop.laracom.net/docs)

# Setup
* `composer install`
* `php artisan migrate --seed`
* `npm install`
* `npm run dev`
* `php artisan storage:link`
